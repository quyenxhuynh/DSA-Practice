## Linked Lists
### Singly-Linked
| Method            | Time-Complexity |
|-------------------|-----------------|
| Insertion (Head)  | O(1)            |
| Insertion (Other) | O(n)            |
| Deletion (Head)   | O(1)            |
| Deletion (Other)  | O(n)            |

### Doubly-Linked
| Method            | Time-Complexity |
|-------------------|-----------------|
| Insertion (Head)  | O(1)            |
| Insertion (Tail)  | O(1)            |
| Insertion (Other) | O(n)            |
| Deletion (Head)   | O(1)            |
| Deletion (Tail)   | O(1)            |
| Deletion (Other)  | O(n)            |
