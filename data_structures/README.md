## Linked Lists
### Singly-Linked
| Method            | Time-Complexity |
|-------------------|-----------------|
| Insertion (Head)  | O(1)            |
| Insertion (Other) | O(n)            |
| Deletion (Head)   | O(1)            |
| Deletion (Other)  | O(n)            |
| Search            | O(n)            |

### Doubly-Linked
| Method            | Time-Complexity |
|-------------------|-----------------|
| Insertion (Head)  | O(1)            |
| Insertion (Tail)  | O(1)            |
| Insertion (Other) | O(n)            |
| Deletion (Head)   | O(1)            |
| Deletion (Tail)   | O(1)            |
| Deletion (Other)  | O(n)            |
| Search            | O(n)            |

### Stack
| Method            | Time-Complexity |
|-------------------|-----------------|
| Push              | O(1)            |
| Pop               | O(1)            |

### Queue
| Method            | Time-Complexity |
|-------------------|-----------------|
| Enqueue           | O(1)            |
| Dequeue           | O(1)            |
